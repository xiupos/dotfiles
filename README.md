# dotfiles

```bash
cd ~
git clone --recursive git@github.com:xiupos/dotfiles.git .dotfiles
cd .dotfiles
make
```

## Guake

```bash
# load prefs
guake --restore-preferences ~/.dotfiles/other/guake_prefs

# save prefs
guake --save-preferences ~/.dotfiles/other/guake_prefs
```
